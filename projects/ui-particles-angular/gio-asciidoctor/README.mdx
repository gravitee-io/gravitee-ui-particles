# Asciidoctor

The Asciidoctor component renders [AsciiDoc](https://asciidoctor.org/docs/what-is-asciidoc/) content using the [Asciidoctor.js](https://asciidoctor.org/docs/asciidoctor-js/) library.

## Installation

This component requires the following peer dependencies as its optional dependencies:

```bash
# asciidoctor
npm install asciidoctor --save
yarn add asciidoctor

# prismjs
npm install prismjs --save
yarn add prismjs
```

Add asciidoctor assets to `angular.json`

```json
...
// projects > [your project name] > architect > build >
"assets": [
    ...
        {
            "glob": "(browser|css)/asciidoctor.(js|css)",
            "input": "node_modules/@asciidoctor/core/dist",
            "output": "assets/asciidoctor"
        }
     ...
    ]
...
```

Add prism and its components assets to `angular.json`

```json
...
// projects > [your project name] > architect > build >
"assets": [
    ...
        {
            "glob": "prism.js",
            "input": "node_modules/prismjs",
            "output": "assets/prismjs"
        },
        {
            "glob": "prism-json.js",
            "input": "node_modules/prismjs/components",
            "output": "assets/prismjs/components"
        }
     ...
    ]
...
```
