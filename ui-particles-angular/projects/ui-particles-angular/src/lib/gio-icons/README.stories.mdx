# Gio Icons

Set of icons used in the screens of the gravitee console.

⚠️ These icons are **always to be preferred before using the material one**.

## Usage

Add the `GioIconsModule` to your application. It will load the gravity icons in material as svg.

To add the icon in the html you have to use mat-icon like :

```html
<mat-icon svgIcon="gio:search"></mat-icon>
```
