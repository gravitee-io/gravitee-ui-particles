# Form Focus invalid

Try to scroll and focus on the first invalid field of a form.

## Usage

Add this directive to the form element.

Example :

```html
<form gioFormFocusInvalid>
  <input type="text" name="name" required />
  <input type="email" name="email" required />
  <input type="submit" />
</form>
```
