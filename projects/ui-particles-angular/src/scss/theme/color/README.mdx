## Use it in SCSS

```scss
@use '@angular/material' as mat;
@use '@gravitee/ui-particles-angular' as gio;

.my-class {
  background-color: mat.m2-get-color-from-palette(gio.$mat-accent-palette);
  color: mat..m2-get-color-from-palette(gio.$mat-accent-palette, contrast);
}

.my-class-lighter {
  background-color: mat..m2-get-color-from-palette(gio.$mat-accent-palette, lighter);
  color: mat..m2-get-color-from-palette(gio.$mat-accent-palette, lighter-contrast);
}
```

### Get Background or Forground color

To get some spesific color only defined into material background or forground

```scss
@use 'sass:map';
@use '@angular/material' as mat;
@use '@gravitee/ui-particles-angular' as gio;

$background: map.get(gio.$mat-theme, background);

.disabled {
  background-color: mat.m2-get-color-from-palette($background, disabled);
}
```
