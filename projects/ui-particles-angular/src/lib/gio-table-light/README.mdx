## Usage

`gio-table-light` is a simple SCSS class that can be used on table element.

```html
<table class="gio-table-light">
  <thead>
      <tr>
          <th>key</th>
          <th>value</th>
      </tr>
  </thead>
  <tbody>
      <tr>
          <td>Accept-Language</td>
          <td>fr,fr-FR;q=0.8,en-US;q=0.5,en;q=0.3</td>
      </tr>
      <tr>
          <td>Accept-Encoding</td>
          <td>gzip, deflate, br</td>
      </tr>
      <tr>
          <td>Cache-Control</td>
          <td>no-cache</td>
      </tr>
      <tr>
          <td>Content-Type</td>
          <td>application/json</td>
      </tr>
  </tbody>
</table>
```
