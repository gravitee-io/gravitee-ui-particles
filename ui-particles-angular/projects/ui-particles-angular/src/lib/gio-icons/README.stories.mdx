# Gio Icons

Set of icons used in the screens of the gravitee console.

‚ö†Ô∏è These icons are **always to be preferred before using the material one**.

## Usage

Add the `GioIconsModule` to your application. It will load the gravity icons in material as svg.

To add the icon in the html you have to use mat-icon like :

```html
<mat-icon svgIcon="gio:search"></mat-icon>
```

## Testing

If you encounter this error `Expected no open requests, found 1: GET assets/gio-icons.svg` you should probably add `MatIconTestingModule` to your test import

## Updating the icons sprite

You can download the sprite.svg file from [Google Drive](https://drive.google.com/drive/folders/1NMoVU7LuICfrG-vLKp9kEJpxFe93ikxi). If you need to update the content of this file, please follow the following steps:

1. Access the Icons folder from [Google Drive](https://drive.google.com/drive/folders/14NQAbGbOLW_PwECyPOkSpl3OFBaWb3Fv?usp=share_link) and download all the icons of the folder.
2. Generate the sprite file by opening the [https://svgsprit.es](https://svgsprit.es/) website, then dragging and dropping the icons into the browser, and finally clicking on `Download Sprite` button on the right.
3. Open a code editor, and do a 'Find & Replace' command: ```Find: fill="#000‚Äù and Replace by: fill="currentColor"``` and ```Find: stroke="#000‚Äù and Replace by: stroke="currentColor"``` üí°¬†*This is to make color overload possible by gravitee CSS*
