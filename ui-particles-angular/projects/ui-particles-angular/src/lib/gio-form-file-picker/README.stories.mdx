# Form File Picker Input

Documentation and examples for Form File Picker input, a component to upload a file.

## Demo

<Story id="components-from-file-picker-input--default"/>

## Usage

**Inputs**
 - `multiple`: `boolean` - Indicates if file picker allows to select multiple files.
 - `accept`: `string` - Type of files accepted by file input. See accepted values [here](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/file#unique_file_type_specifiers). Default is `'*'`
 - `disabled`: `boolean` - Is the input disabled.

Example:

```html
<p>With custom add button</p>
<gio-form-file-picker (ngModelChange)="onChange($event)">
<gio-form-file-picker-add-button><span>Glissez votre fichier</span><p>250x250 px minimum</p></gio-form-file-picker-add-button>
</gio-form-file-picker>


<p>Disabled with a custom empty text</p>
<gio-form-file-picker (ngModelChange)="onChange($event)" [disabled]="true">
<gio-form-file-picker-add-button><span>Glissez votre fichier</span><p>250x250 px minimum</p></gio-form-file-picker-add-button>
<gio-form-file-picker-empty><span class="mat-small">No image defined</span></gio-form-file-picker-empty>
</gio-form-file-picker>

```
